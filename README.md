O História Encantada é uma plataforma educacional de sonoplastia digital
desenvolvida para transformar histórias clássicas em experiências imersivas,
mágicas e interativas.

O projeto utiliza efeitos sonoros organizados por narrativas para auxiliar
na contação de histórias em ambientes educacionais, recreativos e inclusivos.
